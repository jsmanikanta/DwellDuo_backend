# DwellDuo Backend

A robust backend service for the DwellDuo application, built using Node.js, Express.js, and MongoDB. This backend handles user authentication, data management, and API services for a modern web application.

## Features

- RESTful API architecture  
- User authentication (JWT-based)  
- Secure login & registration  
- CRUD operations  
- MongoDB database integration  
- Middleware for authentication & error handling  
- Scalable backend structure
